About My Site

Building this website introduced me to the intricacies of web design,
far exceeding my initial expectations.
Initially, I thought creating a website involved simple codingand basic layouts,
but I soon learned the versatility of HTML in structuring content.


The integration of CSS with HTML was transformative,
allowing for dynamic, visually appealing sites.
My earlier understanding of CSS was superficial;
now, I see its profound impact on a site’s aesthetics and user interaction.
Learning to use JavaScript enhanced the site's interactivity,
making static pages come alive,
which underscored the importance of user experience design.


Looking back, I'd spend more time initially planning the site's structure.
Understanding frameworks like React earlier could have streamlined development.
This project wasn’t just about technical skills;
it was a comprehensive learning journey
that expanded my understanding of web development
and sparked further curiosity about advanced tools and techniques.
